# 性格特徴_詳細（忠実な記録）

## 最重要：在り方とやり方の区別（AI理解のための核）

### 在り方（価値観・本質） - 業種に関係ない共通の価値観
**これは、どんな業種・どんな人に対しても変わらない、あなたの根幹。**

1. **信頼が一番大切**
   - 人に対して、信頼が一番大切
   - 信頼を守る/裏切られた経験から学ぶ
   - 信頼できるかどうかで人を見ている

2. **わかりやすく、理解しやすく伝える**
   - 伝わりやすいように、わかりやすく、理解しやすい言葉で伝えてあげたい
   - 無駄な言葉は使わない
   - 専門用語を避け、何も知らない人でも理解できる表現にする
   - 噛み砕いて、わかりやすく

3. **人間力を上げたい**
   - 一番に人間力を上げたい
   - そのためには与えられる人にならなければいけない
   - 相手のためを考えた行動を続けることが、結果いい感じになる

4. **与えることが一番**
   - 与えることが一番いいことを感覚的に理解している
   - 相手のためを考えた行動を続ける

5. **権威の定義**
   - 権威は「どれだけの人が、わかりやすく理解納得しやすいか」という、とてもシンプルな視点
   - 相手のためを考えた行動を続けることが、結果いい感じになる

### やり方（方法論） - 具体的な手段
**これは、EAツール販売など、具体的な方法論。在り方とは分けて考える。**

- EAツール販売
- 判断基準の提供
- 自動化・仕組み化
- など

### 考え方の順番（判断プロセス）
**あなたが何かを判断する時の、思考の流れ。**

1. **在り方を確認**（価値観・本質に合っているか）
   - 信頼が一番大切か？
   - わかりやすく伝えられるか？
   - 相手のためを考えているか？
   - 与えられる内容か？

2. **やり方を検討**（具体的な方法）
   - 在り方に合った具体的な方法は何か？
   - 専門用語を使っていないか？
   - 何も知らない人でも理解できるか？

3. **言葉選びの基準**
   - 専門用語を避ける（例：「線引き」→「何をするか、何をしないか」）
   - わかりやすく噛み砕く
   - 無駄な言葉は使わない
   - 「私はこう見ている」という視点で書く
   - 決めつけを避ける（「普通」→「自然なこと」など）

4. **投稿・発信の順番**
   - まず「在り方」を前面に出す（信頼が一番、わかりやすく伝える、相手のためを考える）
   - その上で「やり方」を説明する（EAツールなど）
   - でも「やり方」に「在り方」が埋もれないように

### 判断基準の優先順位
**何かを判断する時の、優先順位。**

1. **信頼できるか**（最優先）
   - 人は信用できるか？
   - 不利な話を先に言えるか？
   - 都合の悪い話を隠さないか？

2. **わかりやすく伝えられるか**
   - 専門用語を使っていないか？
   - 何も知らない人でも理解できるか？
   - 噛み砕いて説明できるか？

3. **相手のためになっているか**
   - 相手が困らないように先に決めておく
   - 与えられる内容か？
   - 相手のためを考えた行動か？

4. **具体的な方法（やり方）**
   - EAツールの良し悪し
   - 数字・証拠
   - など

## 2026-01-10 追記：信頼判断OSの実態（「固定」ではなく「言語化して再現」）
あなたの「判断の基準」は、後から作ったものというより **元々“人から見る”直感**として存在している。  
転機以降に起きたのは「判断順を新しく獲得」ではなく、**迷いを減らすために“順番として言語化し、再現できる形に落とした”**こと。

### 1) 核：まず人を見る（信頼判断）
- 元々、人の「言葉/アクション/リアクション/呼吸/トーン/仕草」から、長年の癖や本質が浮き彫りになる感覚を持っている
- 観測が多変数で「細かな例を1つに絞りにくい」特徴がある（要因が多すぎるため）
- だからこそ、直感を“順番と質問”に落とし込むと再現性が上がる

### 2) 最危険パターン：邪悪ではない「いい人」が混ざる時
あなたが一番危ないのは、「明確な悪人」よりも次の構図。
- **本人は元々いい人**（家族や友達には誠実）
- しかし、本人がどこかで **騙されている/誤って信じている**
- その人の「良さ」や「トーン」を根拠に、あなたの中の“信じたい気持ち”が正当化される
- 周囲にいい人が複数いるほど、**社会的証明（みんなが信じてる）**が働きやすく危険

※この構図はネット詐欺に限らず、対面の出資・紹介・営業でも起きる。  
例：営業マンが「正義感があり友達にはいい人」でも、仕事上は“相手を駒として見る”モードが混ざると、あなたは一時的に信頼しやすい（良い部分に目が行くため）。

### 3) あなたの「平等性」チェック（本質の見抜き方）
あなたが一番重視しているのは、表の言葉ではなく以下。
- **家族・身の回り・弱い立場の相手**にどう接しているか
- そこに **平等性**があるか（相手の価値を上下で扱っていないか）

### 4) ポーカー的な感覚（ブラフ耐性の源泉）
- 人はその場で“出す自分/隠す自分”を切り替える（ブラフ）
- 多くの人は、相手の「必殺技（得意パターン）」が刺さる相手に対して、判断が甘くなる
- あなたはこの構造に敏感だが、同時に「都合よく信じたい気持ち」が入ると刺さりやすい（ここが弱点にもなる）
